# Section 1: AZ-900 Exam Overview

**Exam:** AZ-900 Microsoft Azure Fundamentals
**Score:** 700/1000 to pass | 40-60 questions | 45 minutes | $99 USD

## Three Exam Domains

| Domain | Weight |
|--------|--------|
| Cloud Concepts | 25-30% |
| Azure Architecture and Services | 35-40% |
| Azure Management and Governance | 30-35% |

Domains 2 and 3 together make up 65-75% of the exam. Focus your study time there.

## Key Exam Tips

- "MOST cost-effective" = look for the cheapest option
- "LEAST operational overhead" = look for managed/PaaS/serverless
- "High availability" = availability zones, region pairs, scale sets
- "NOT" = read carefully, they test what does NOT apply
- Don't confuse Entra ID with traditional Active Directory
- Don't confuse Pricing Calculator with TCO Calculator
- Don't confuse Azure Policy with RBAC
