## Why Multiple Sources Per Certification

Certification exams test knowledge in isolation. Real security work requires connecting concepts across domains. These notes deliberately cross-reference:

**AZ-900** integrates DevOps pipeline concepts (how cloud enables CI/CD), shell scripting examples (how you actually manage Azure from the terminal), and Thales breach data (why shared responsibility and MFA gaps matter in the real world).

**AZ-500** integrates the CSA Security Guidance framework (vendor-neutral cloud security principles behind everything Microsoft-specific) and Thales threat landscape data (real-world context for every control you configure).
